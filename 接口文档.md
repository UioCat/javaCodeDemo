# 接口文档
[toc]

根路径
```
http://127.0.0.1:8080/demo
```

## 注册接口
url
```
/register
```
req
```json
{
    "telephone":"123456120",
    "username":"xiaoming",
    "password":"123456"
}
```

res
```json
{
    "code": 200,
    "message": "请求成功",
    "info": null
}
```

## 登陆接口
url
```
/login
```
req
```json
{
    "telephone":"123456120",
    "password":"123456"
}
```
res
```json
{
    "code": 200,
    "message": "请求成功",
    "info": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJJRCI6MjAsImV4cCI6MTYzNjc0NDExOX0.118Ie4tvkVJFo6ct8FBz-BTK23DH7X00s9UjCd8AlyU"

```
